# Roadmap Profissional — Rômulo Sacramento

Este roadmap organiza minha evolução técnica e profissional com foco em suporte corporativo, ERP, processos de concessionárias, automação e arquitetura de soluções.

---

## 1. Fundamentos de TI

Objetivo: consolidar a base para atuar com suporte L1/L2 e diagnóstico técnico.

### Estudar e praticar

- Troubleshooting estruturado
- Windows corporativo
- TCP/IP, DNS, DHCP e Gateway
- Comandos: `ipconfig`, `ping`, `tracert`, `nslookup`, `netstat`
- Active Directory
- Microsoft 365
- ITIL básico
- NOC e monitoramento

### Entregáveis

- Caderno de fundamentos
- Laboratório de suporte
- Banco de problemas resolvidos
- Checklists operacionais

## Princípio de estudo

Todo novo assunto deve ser estudado nesta sequência:

```text
Negócio → Processo → Regra de Negócio → Regra Funcional → Sistema → Integração → Banco de Dados → Suporte → Melhoria Contínua
```
